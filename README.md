## 描述

我基于 [Nest](https://github.com/nestjs/nest) 对开发的我的毕设项目后端的另一个版本，之前开发了一个[Egg的版本](https://github.com/cshenger/mykms-backed-egg)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
